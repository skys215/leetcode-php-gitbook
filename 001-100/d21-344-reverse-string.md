# D21 344. Reverse String

## 题目链接

[344. Reverse String](https://leetcode.com/problems/reverse-string)

## 题目分析

题目要求以O\(1\)时间复杂度把字符串倒转过来。

## 思路

题目提示说用原地算法……Emmm...我并不会，只能用`strrev`函数先应付了。

## 最终代码

```php
<?php
class Solution {
    function reverseString($s) {
        return strrev($s);
    }
}
```
